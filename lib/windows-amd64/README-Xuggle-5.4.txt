Xuggle-5.4 has instability problems on Windows 64bit platform.
See thread
    http://www.emunewz.net/forum/showthread.php?tid=42305
for more details.

Workaround: use Xuggle-5.4 for all platforms, excepted for windows-amd64.
    Use Xuggle-3.4 for windows-amd64.
    "build-auto.xml" is packaging Xuggle-3.4 for windows-amd64 instead of Xuggle-5.4
